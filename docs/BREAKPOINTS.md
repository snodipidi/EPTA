# Брейкпоинты — ЕПТА

## Мобильные (< 1024px)

Файл: `frontend/src/styles/mobile.css` (импорт в `global.css`).

| Элемент | Поведение |
|---------|-----------|
| Шапка | Только поиск на всю ширину, логотип скрыт |
| Sidebar | Drawer слева, открывается поверх контента (гамбургер в нижней панели) |
| Нижняя панель | Меню · Главная · Уведомления · Профиль |
| Контент | Одна колонка, `width: 100%`, отступ снизу под nav |
| Auth | Логотип сверху, форма снизу |
| Профиль | Вертикальная карточка, кнопка входа/выхода на всю ширину |

**Новые страницы:** рендерить внутри `Layout` → контент попадает в `.layout__content`.
Корень страницы: `width: 100%; min-width: 0;`. Мобильные стили добавлять в `mobile.css`.

## Десктоп (≥ 1024px)

Минимальная ширина окна — `1024px`. Адаптив под мониторы.

### Как работает ширина

Контент **плавно сжимается** вместе с окном, а не только на брейкпоинтах:

```css
--layout-width: min(80vw, var(--layout-max-width));
```

- При уменьшении окна сужаются и шапка, и лента, и посты
- `minmax(0, 1fr)` в сетке — колонка ленты может сжиматься
- `min-width: 0` на карточках — текст не раздувает блок

### Таблица брейкпоинтов

| Имя | min-width | Layout max | Fluid width | Gap (sidebar ↔ feed) |
|-----|-----------|------------|-------------|----------------------|
| `sm` (base) | **1024px** | 820px | `min(80vw, 820px)` | `clamp(28px, 4vw, 48px)` |
| `md` | **1280px** | 920px | `min(82vw, 920px)` | `clamp(36px, 4.5vw, 56px)` |
| `lg` | **1440px** | 1000px | `min(84vw, 1000px)` | `clamp(40px, 5vw, 64px)` |
| `xl` | **1920px** | 1100px | `min(86vw, 1100px)` | `clamp(48px, 5vw, 72px)` |

### CSS-переменные

Файл: `frontend/src/styles/variables.css`

```css
--layout-width          /* реальная ширина блока (fluid) */
--layout-max-width      /* верхний предел на брейкпоинте */
--page-padding-x        /* отступы слева/справа от края окна */
--sidebar-feed-gap      /* зазор между меню и лентой */
--header-toolbar-gap    /* зазор между поиском и иконками */
--mobile-nav-height     /* высота нижней панели (mobile) */
```
