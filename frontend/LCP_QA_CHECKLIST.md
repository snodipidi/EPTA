# Чек-лист для QA: Проверка оптимизации LCP

## Перед началом тестирования

### ✅ Подготовка среды
- [ ] Установить последнюю версию Chrome (≥ 116)
- [ ] Очистить кэш браузера (Ctrl+Shift+Delete)
- [ ] Отключить расширения, влияющие на загрузку
- [ ] Использовать режим инкогнито для чистого тестирования

## Базовые проверки

### ✅ 1. Сборка проекта
```bash
npm run build        # Должна завершиться без ошибок
npm run preview      # Должен запуститься на localhost:4173
```

### ✅ 2. Проверка типов TypeScript
```bash
npx tsc --noEmit     # Не должно быть ошибок компиляции
```

## Функциональные проверки

### ✅ 3. Рендеринг ленты
- [ ] Главная страница загружается без ошибок
- [ ] Посты отображаются корректно
- [ ] Изображения в постах загружаются
- [ ] Не должно быть сообщений об ошибках в консоли

### ✅ 4. Проверка атрибутов изображений
Откройте DevTools > Elements и проверьте:

**Для первого поста (первое изображение):**
- [ ] `fetchpriority="high"` присутствует
- [ ] `loading="eager"` или атрибут отсутствует
- [ ] `width` и `height` заданы числовыми значениями

**Для остальных постов:**
- [ ] `loading="lazy"` присутствует
- [ ] `width` и `height` заданы числовыми значениями
- [ ] `fetchpriority` отсутствует

## Производительность

### ✅ 5. Network Panel проверки
1. Откройте DevTools > Network
2. Перезагрузите страницу (Ctrl+R)
3. Проверьте:

**Фильтр по Type: img**
- [ ] Первое изображение: Priority = **High**
- [ ] Остальные изображения: Priority = **Low**
- [ ] Изображения загружаются с правильными размерами

### ✅ 6. Performance Panel тестирование
1. Откройте DevTools > Performance
2. Нажмите "Record"
3. Перезагрузите страницу
4. Дождитесь полной загрузки
5. Остановите запись
6. Проверьте:

- [ ] Largest Contentful Paint метка присутствует
- [ ] Время LCP отображается
- [ ] Нет длительных задач блокирующих рендеринг

### ✅ 7. Lighthouse Audit
1. Откройте DevTools > Lighthouse
2. Настройки:
   - Device: Mobile + Desktop
   - Categories: Performance
   - Clear storage: включено
3. Запустите анализ
4. Проверьте результаты:

**Performance Score:**
- [ ] Mobile: ≥ 90 (желательно)
- [ ] Desktop: ≥ 95 (желательно)

**Core Web Vitals:**
- [ ] LCP: ≤ 2.5 секунд (зеленая зона)
- [ ] CLS: ≤ 0.1 (зеленая зона)
- [ ] FID: ≤ 100 мс (зеленая зона)

### ✅ 8. Layout Stability (CLS)
- [ ] При загрузке страницы нет прыжков контента
- [ ] Изображения не смещают другие элементы
- [ ] Все изображения имеют резервированное место

## Специфические сценарии

### ✅ 9. Тестирование на разных устройствах
- [ ] Мобильный телефон (эмуляция в DevTools)
- [ ] Планшет (768px)
- [ ] Десктоп (≥ 1200px)
- [ ] Плохое соединение (Slow 3G в DevTools)

### ✅ 10. Поведение при скролле
- [ ] Lazy изображения загружаются при приближении к viewport
- [ ] Нет "моргания" при загрузке lazy изображений
- [ ] Плавный скролл без задержек

## Edge Cases

### ✅ 11. Граничные случаи
- [ ] Страница без изображений (должна работать)
- [ ] Пост с одним изображением
- [ ] Пост с несколькими изображениями
- [ ] Пост без width/height в данных (должен handle gracefully)
- [ ] Быстрый скролл во время загрузки

### ✅ 12. Повторные посещения
- [ ] После первого посещения (кеширование работает)
- [ ] Hard reload (Ctrl+Shift+R) очищает кэш
- [ ] Service Worker (если есть) не мешает оптимизациям

## Мониторинг и метрики

### ✅ 13. Консольные проверки
```javascript
// В DevTools Console выполните:
const images = document.querySelectorAll('img');
console.log('Всего изображений:', images.length);

const lcpCandidate = document.querySelector('img[fetchpriority="high"]');
console.log('LCP кандидат:', lcpCandidate?.src);

// Проверка атрибутов
images.forEach((img, i) => {
  console.log(`Изображение ${i}:`, {
    src: img.src,
    width: img.width,
    height: img.height,
    loading: img.loading,
    fetchPriority: img.fetchPriority,
    complete: img.complete
  });
});
```

### ✅ 14. Performance API метрики
```javascript
// Проверка метрик загрузки
if ('performance' in window) {
  const paint = performance.getEntriesByType('paint');
  console.log('Paint метрики:', paint);
  
  const lcpEntries = performance.getEntriesByType('largest-contentful-paint');
  console.log('LCP метрики:', lcpEntries);
}
```

## Критерии успеха

### Основные (должны быть выполнены все):
- [ ] LCP ≤ 2.5 секунд на мобильных устройствах
- [ ] LCP элемент загружается с приоритетом High
- [ ] Нет Cumulative Layout Shift (CLS = 0)
- [ ] Все изображения имеют width и height атрибуты

### Дополнительные (рекомендуемые):
- [ ] Lighthouse Performance Score ≥ 90
- [ ] FID ≤ 100 мс
- [ ] Smooth scrolling при lazy loading
- [ ] Корректная работа на Slow 3G
