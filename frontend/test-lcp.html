Инструкции для тестирования:

1. Использование Chrome DevTools:

  • Откройте DevTools (F12)
  • Перейдите во вкладку "Performance"
  • Нажмите "Record" и перезагрузите страницу
  • Найдите метку "Largest Contentful Paint" на временной шкале

2. Использование Network Panel:

  • Откройте вкладку "Network"
  • Перезагрузите страницу (Ctrl+R)
  • Проверьте колонку "Priority" - LCP элемент должен иметь "High"
  • Остальные изображения должны иметь "Low"

3. Lighthouse Audit:

  • Откройте DevTools
  • Перейдите во вкладку "Lighthouse"
  • Выберите "Mobile" и "Desktop"
  • Нажмите "Analyze page load"
  • Проверьте показатель LCP в разделе "Performance"

Целевые показатели:

  • LCP ≤ 2.5 секунд - Отлично (зеленая зона)
  • LCP 2.5–4.0 секунд - Нужны улучшения (желтая зона)
  • LCP > 4.0 секунд - Плохо (красная зона)
